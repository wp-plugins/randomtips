== Installation ==

This section describes how to install the plugin and get it working.

1. Create folder `/wp-content/plugins/randomtips`
2. Unzip the plugin into `/wp-content/plugins/randomtips` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Place [randomtips] in your posts or pages.

== Frequently Asked Questions ==

= How to insert new tip? =

From Settings>RandomTip compile the form to insert a new tip. Compile "Text" field to set the text to show - it support HTML markup -
compile the group field to specify the group in which you'd like to collate your tips. Please note that in case no group is specified 
"default" group is used.

= How should I use the "group" field? =

"Group" field is used to create group of tips to display. using the shortcode you can specify which group to show
[randomtips group=mygroup] in case no group is specified the default will be used [randomtips].
